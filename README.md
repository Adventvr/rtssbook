## Gitbook 指南
* 安装命令行工具: `npm install -g gitbook-cli` 建议使用最新 3.0 以上版本.
* 编译前先安装 `book.json` 中配置的插件.
* 本地编译: `gitbook build` 会将 build 后网页文件放到 `_book` 目录.
* 本地查看: `gitbook serve` 默认会进行 build, 然后 serve 一个网页提供给本地浏览器访问.
